Build a Node.js + Express.js backend for "TradeSignal Pro" algo trading app.

== ANGELONE SMARTAPI CONFIG ==
BASE_URL = https://apiconnect.angelone.in
API VERSION: v1 (all routes use v1)
Required headers on ALL requests:
{ Authorization: "Bearer {jwtToken}", Content-Type: "application/json",
  X-UserType: "USER", X-SourceID: "WEB", X-ClientLocalIP: "...",
  X-ClientPublicIP: "...", X-MACAddress: "00:00:00:00:00:00",
  X-PrivateKey: "{API_KEY}" }

== AUTH ENDPOINTS ==
LOGIN: POST /rest/auth/angelbroking/user/v1/loginByPassword
  Body: { clientcode, password, totp }
  Returns: { jwtToken, refreshToken, feedToken }
REFRESH: POST /rest/auth/angelbroking/jwt/v1/generateTokens
  Body: { refreshToken }
LOGOUT: POST /rest/secure/angelbroking/user/v1/logout
TOTP: Generate with otplib → authenticator.generate(TOTP_SECRET)
Auto-refresh jwtToken 30 mins before expiry using refreshToken.

== USER & FUNDS ==

GET Funds/Margin: GET /rest/secure/angelbroking/user/v1/getRMS

== MARKET DATA ==
Live Quote: POST /rest/secure/angelbroking/market/v1/quote/
  Body: { mode:"LTP|OHLC|FULL", exchangeTokens:{ NSE:["2885","1594"] } }
  FULL returns: ltp, open, high, low, close, volume, percentchange, 52wHigh,
  52wLow, upperCircuit, lowerCircuit, bid/ask depth (5 levels)
  Max 50 symbols per call.

Historical Candles: POST /rest/secure/angelbroking/historical/v1/getCandleData
  Body: { exchange, symboltoken, interval, fromdate:"YYYY-MM-DD HH:MM", todate }
  Intervals: ONE_MINUTE | THREE_MINUTE | FIVE_MINUTE | TEN_MINUTE |
             FIFTEEN_MINUTE | THIRTY_MINUTE | ONE_HOUR | ONE_DAY
  Response array: [timestamp, open, high, low, close, volume]
  Transform: { time: unixTimestamp, open, high, low, close, volume }
  Limits: 1-5min=30days, 15-30min=90days, 1H-1D=365days per request

Symbol Search: POST /rest/secure/angelbroking/order/v1/searchScrip
  Body: { exchange:"NSE", searchscrip:"RELIANCE" }
  Returns: tradingsymbol, symboltoken, instrumenttype, name

Gainers/Losers: GET /rest/secure/angelbroking/marketData/v1/gainersLosers
  ?datatype=PercPriceGainers&expirytype=NEAR

== WEBSOCKET LIVE DATA ==
URL: wss://smartapisocket.angelone.in/smart-stream (authenticate with feedToken)
Subscribe payload: { action:1, params:{ mode:3, tokenList:[{exchangeType:1, tokens:["2885"]}] }}
exchangeType: 1=NSE_CM, 2=NSE_FO, 3=BSE_CM, 5=MCX_FO | mode: 1=LTP, 2=Quote, 3=Full
Heartbeat: send { ping:"ping" } every 30s
Auto-reconnect with exponential backoff (3s, 6s, 12s...) on disconnect.
Broadcast ticks to frontend via SSE: GET /api/live/stream

== ERROR HANDLING ==
AB1010 = JWT expired → auto-call generateTokens
AB1004 = Invalid API key → notify user
AB8000 = Rate limit → backoff 60s
AB1005 = Session not found → re-login
Rate limits: Orders=5 req/s, Historical=3 req/s, Quote=1 req/s
Use bottleneck npm for rate limiting: new Bottleneck({ minTime:200 })

== SIGNAL SCORING ENGINE ==
Score each condition, sum total, classify:
  EMA9>EMA21 +1 | EMA21>EMA50 +1 | EMA50>EMA200 +2 | Price>EMA200 +2
  EMA9<EMA21 -1 | EMA50<EMA200 -2 | Price<EMA200 -2
  MACD BullCross +3 | BearCross -3 | Histogram rising +1
  Supertrend BUY +2 | SELL -2
  RSI<25 +3 | <30 +2 | <40 +1 | >75 -3 | >70 -2 | >60 -1
  Bull RSI Divergence +3 | Bear -3
  Stoch BullCross oversold +2 | BearCross overbought -2
  Volume 2x+ green +2 | red -2 | 1.5x+ +1
  Price>VWAP +1 | <VWAP -1 | MFI<20 +2 | >80 -2
  BB lower touch +2 | upper touch -2 | squeeze → note only
  At support +2 | breakout above resistance +3 | breakdown below -3
  Candle patterns: hammer+2, bullEngulf+2, morningStar+3, 3WhiteSoldiers+3
    shootingStar-2, bearEngulf-2, eveningStar-3, 3BlackCrows-3
  Chart patterns: doubleBottom+3, cupHandle+3, fallingWedge+2, invH&S+3
    doubleTop-3, H&S-3, risingWedge-2
  ADX>25 +DI>-DI +1 | -DI>+DI -1

Classification:
  ≥12 = STRONG_BUY | 7-11 = BUY | 3-6 = WEAK_BUY
  -2 to 2 = NEUTRAL | -3 to -6 = WEAK_SELL | -7 to -11 = SELL | ≤-12 = STRONG_SELL
  confidence = min(95, max(10, 50 + score*3))

Risk Management:
  SL_buy = min(last20Lows)*0.995 or currentPrice-(ATR*1.5)
  T1 = entry+(entry-SL)*1.5 | T2 = entry+(entry-SL)*2.5 | T3 = entry+(entry-SL)*3.5
  If R:R < 1.5 → override to NEUTRAL, push reason "R:R too low"

== PAPER TRADING ENGINE ==

MARKET orders fill at current LTP instantly.
LIMIT orders fill when LTP crosses limit (poll every 5s).
SL triggers monitored continuously.
Virtual balance ₹10,00,000. Track in memory/localStorage per session.
GET /api/paper/portfolio → simulated portfolio + P&L
