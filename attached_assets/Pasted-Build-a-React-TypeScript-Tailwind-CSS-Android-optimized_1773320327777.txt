Build a React + TypeScript + Tailwind CSS Android-optimized web app
"TradeSignal Pro" — a professional algo trading signal app.
Backend API base: http://localhost:3000

== BRANDING & THEME ==
Dark mode only. Colors:
  bg: #0A0A0F | card: #12121A | border: #1E1E2E | green: #00FF88
  red: #FF3366 | blue: #00BFFF | gold: #FFD700 | text: #EAEAEA
  muted: #8B8B9E | inputBg: #1A1A2E | warning: #FF9500
Watermark EVERY screen bottom: "Made with ❤️ by Shahrukh"
  Style: 11px, color #555555, ❤️ red #FF0000, centered, thin separator above.

== ANDROID SCREEN RULES (CRITICAL) ==
Viewport: 360-412px width. Mobile-first CSS only.
Safe area: top 28px (status bar), bottom 64px (nav bar).
Touch targets: min 48×48px. Text: min 13px (body 14-16px).
Cards: border-radius 14px, padding 16px, margin 8px, shadow 0 2px 8px rgba(0,0,0,0.3)
Buttons: full-width, height 50px, border-radius 12px, font-weight 600.
Inputs: full-width, height 48px, border-radius 10px, font-size 16px.
Scrolling: -webkit-overflow-scrolling: touch. NO horizontal scroll.
Transitions: 200ms slide/fade. Pull-to-refresh on data screens.

== SCREEN 2: SETTINGS ==
Scrollable. 6 sections:

SECTION 1 — BROKER CONNECTION
Card: "Connect Broker"
Toggle tabs: AngelOne | Groww
  AngelOne inputs: API Key, PIN, TOTP Secret (auto-generates live TOTP preview with 30s countdown)
  Groww sub-tabs: API Key+Secret | TOTP Flow | Direct Token
  Each input has eye toggle for show/hide.
Button: "🔌 Validate & Connect" (full-width blue)
LIVE validation stepper (vertical timeline):
  Step 1: Format check ✓/✗
  Step 2: Generate token (spinner → success/error with message)
  Step 3: Connect to servers (test profile API call)
  Step 4: Fetch account details (show name, client ID)
  Step 5: Test market data (fetch RELIANCE quote live)
  Step 6: Check order permissions
CONNECTION STATUS CARD (updates live):
  🟢 CONNECTED: name, client ID, token expiry countdown, "Re-validate | Disconnect"
  🟡 PARTIAL: connected but limited features
  🔴 FAILED: error message + troubleshooting steps + "Try Again"
Auto-revalidate every 5 minutes silently. Show token expiry countdown.
30 mins before expiry: banner "Token expiring soon. Tap to refresh."

SECTION 2 — GEMINI AI
Card: "🤖 Gemini AI Configuration"
Input: API Key (eye toggle, char count ~39)
Expandable help: "How to get free key" → step-by-step guide → link to aistudio.google.com
Model dropdown: gemini-2.0-flash (default) | gemini-1.5-pro | gemini-1.5-flash
Analysis depth cards: Quick(5s) | Standard(15s, default) | Deep(30s)
Button: "🧪 Test Gemini" → validation stepper:
  Step 1: Key format check (starts with AIzaSy, ~39 chars)
  Step 2: Real API call → POST .../gemini-2.0-flash:generateContent?key=...
    { contents:[{parts:[{text:"Reply CONNECTED"}]}] }
  Step 3: Test analysis capability (send sample stock data prompt)
  Step 4: Show rate limit remaining
STATUS CARD:
  🟢 Active: model, rate limit, "AI ready!" | 🔴 Failed: error + fix guide
  🔷 Not configured: "Add free key above. 60 req/min, no credit card!"
Auto health-check every 10 mins.

Optional section (collapsible): Add OpenAI (sk-...) or Claude (sk-ant-...) as alternative.

SECTION 3 — TRADING PREFERENCES
Dropdowns: Trading Mode, Segment, Exchange, Product Type, Order Type, Risk Level
Number inputs: Max Loss% (default 2), Target% (default 4), Max Daily Loss% (5),
  Max Trades/day (10), Default Quantity (1)
Dropdown: Preferred Timeframe (default 15m)

SECTION 4 — ALERTS & SAFETY
Toggles (with descriptions):
  Auto Stop Loss ON | Trailing Stop Loss ON | Sound Alerts ON
  Vibration ON | Auto Trade OFF (red warning if ON) | Paper Trading ON | Detailed Analysis ON

SECTION 5 — CONNECTION DASHBOARD (top of settings page)
Summary card always visible:
  Groww 🟢/🔴 | Gemini AI 🟢/🔴 | Market 🟢/🔴 | Paper Mode badge
  "Everything ready!" or specific issue message.
  Dots are tappable — show detail tooltip.

Save button: "💾 Save All Settings" (green). Reset link below.

AI ANALYSIS BUTTON (large, prominent, gold gradient border):
  Label: "🤖 Analyze with Gemini AI"
  Shows badge: "AI Ready ✅" or "Setup Needed ⚠️"
  If not configured → modal: "Get free Gemini key" with link + "Go to Settings"
  If configured → tap shows full-screen overlay:
    Dark backdrop + animated AI brain icon + progress steps:
    "Collecting 200 candles..." → "Calculating indicators..." →
    "Detecting patterns..." → "Sending to Gemini..." → "Parsing response..."
  
  Send to Gemini:
    POST https://generativelanguage.googleapis.com/v1beta/models/{model}:generateContent?key={apiKey}
    Body: { contents:[{parts:[{text: fullPrompt}]}], generationConfig:{temperature:0.3, maxOutputTokens:4096} }
    fullPrompt includes: stock, timeframe, last 100 OHLCV candles (JSON),
      all indicator values, detected patterns, support/resistance levels.
    Ask Gemini to return strict JSON: { overall_trend, signal, confidence_percentage,
      entry_price, target_1, target_2, target_3, stop_loss, risk_reward_ratio,
      time_horizon, reasoning[], risk_factors[], chart_pattern_analysis,
      volume_analysis, momentum_analysis, summary }
    Extract JSON from response (handle markdown code block wrapping).
    Error handling: 429→show 60s countdown retry | 401/403→"Invalid key, go to Settings"

  AI Result Card:
    Large signal circle (🟢/🔴/🟡) + confidence ring (% in center)
    Trade Setup: Entry ₹X | T1 ₹X (+X%) | T2 ₹X | T3 ₹X | SL ₹X | R:R 1:X
    "Why?" section (green bullets, expandable) + "Risks" (red bullets, expandable)
    Detailed Analysis paragraphs (expandable)
    Buttons: Copy Analysis | Re-analyze | Execute Trade
    Counter: "Gemini analyses today: 24/1500"
